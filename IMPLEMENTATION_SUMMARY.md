## 🔧 核心算法实现详情

### 短线策略算法（基于技术指标）

#### 策略①: 均线回踩低吸策略
```python
# 策略逻辑
1. 股价在20日线上方
2. 20日线趋势向上（MA20 > 前10日MA20）
3. 回踩5/10日线（价格接近均线2-3%）
4. 成交量验证：放量上涨→缩量回踩
5. K线形态：小阳线或十字星企稳

# 评分计算
基础分0.7 + 成交量得分(0.2) + K线形态得分(0.1)

# 风险控制
买入价：当前收盘价
止损价：-7%（买入价×0.93）
止盈价：+8%（买入价×1.08）
```

#### 策略②: 突破缩量回踩策略
```python
# 策略逻辑
1. 识别箱体震荡区间（15-30日）
2. 检测放量突破箱体上沿
3. 确认缩量回踩原压力位转支撑
4. 收盘价站稳支撑线

# 评分计算
基础分0.65 + 成交量模式得分

# 风险控制
止损价：-7%或箱体下沿（取较小值）
止盈价：+12%（突破后目标涨幅）
```

#### 策略③: 强势股10日线反抽策略
```python
# 策略逻辑
1. 强势股筛选：近20日涨幅 > 大盘
2. 第一次回踩10日线（最近20日无其他回踩）
3. 回踩收阳线，出现承接迹象
4. 成交量：回调缩量→反弹放量

# 评分计算
基础分0.68 + 涨幅得分 + 成交量得分

# 风险控制
止损价：-7%
止盈价：+10%（反弹至前高附近）
```

### 中线策略算法（技术面实现）

#### 策略①: 行业成长均线多头
```python
# 策略逻辑
1. 均线多头排列：5>10>20>60
2. 突破后回踩关键均线企稳
3. 行业景气度（技术面替代）

# 风险控制
止损价：-12%（中线容忍度更高）
止盈价：+25%（中线目标收益）
```

#### 策略②: 困境反转策略
```python
# 策略逻辑
1. 前期调整幅度 >30%
2. 近期放量突破60日线
3. 技术面反转信号

# 风险控制
止损价：-10%
止盈价：+20%（估值回归）
```


### 3. **API接口使用示例**

#### 获取策略列表
```bash
GET /api/strategies

响应:
```

## ⚠️ 当前限制和注意事项

### 1. **数据源限制**
- **财务报表数据**: 当前数据源无法获取，基本面策略暂不实现
- **实时数据延迟**: 免费数据源有15分钟延迟

### 2. **策略局限性**
- **纯技术面分析**: 缺乏基本面支撑
- **参数敏感性**: 策略效果依赖阈值参数
- **市场适应性**: 不同市场环境下策略表现差异

### 3. **风险提示**
- **仅供研究使用**: 不构成投资建议

- **风险自担**: 投资有风险，决策需谨慎
